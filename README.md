# Product Management App

A full-stack Product Management application built using
React, Material UI, Node.js, Express, and MongoDB.

## Features
- User Authentication (Signup / Login)
- Product CRUD operations
- Responsive UI using Material UI
- RESTful APIs

## Tech Stack
Frontend:
- React
- Material UI
- Axios


Backend:
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Installation

### Backend
```bash
cd backend
npm install
